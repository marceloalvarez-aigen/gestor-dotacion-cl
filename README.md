# gestor-dotacion-cl

Sistema de gestión de dotación docente conforme a Ley 20.903 de Chile.

## Características

- Gestión de establecimientos educacionales
- Registro de funcionarios docentes
- Control de contratos y carga horaria
- Reportes para Mineduc

## Tecnologías

- React + TypeScript
- Supabase (Backend y Auth)
- Vite
- React Router
